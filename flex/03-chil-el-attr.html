예를들어 자식요소 3개에 flex-grow를 각각 1, 2, 1이라면 너비가 비율에 따라 25%(1/4), 50%(1/2), 25%(1/4)가 배정된다.

자식요소의 너비(증가, 감소, 기본)을 설정하는 단축 속성 flex: [증가너비] [감소너비] [기본너비]

    .child {
      flex: 1 1 50px; 증갑너비 감소너비 기본너비
      flex: 1 1;      증가너비 감소너비
      flex: 1 50px;   증가너비 기본너비
    }
  • flex 단축속성에서 flex-grow 를 제외하고는 모두 생략 가능하다
  • flex: 1; == flex-grow: 1;
  • flex: 1; == flex: 1 1; == flex: 1 1 0;

flex: 숫자(정수)
