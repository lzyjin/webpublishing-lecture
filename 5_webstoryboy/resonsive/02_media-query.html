미디어쿼리

@media 규칙은 미디어 쿼리에서 서로 다른 미디어 유형/디바이스에 서로 다른 스타일을 적용하는 데 사용된다.

미디어 쿼리를 사용하는 것은 데스크롭, 랩톱, 태블릿 및 휴대폰에 맞춤형 스타일 시트(responsive web design)를 제공하는 데 널리 사용되는 기술이다.

또한 미디어 쿼리를 사용하여 특정 스타일이 인쇄된 문서 또는 스크린리더(미디어 유형: print, screen, or speech)에만 적용되도록 지정할 수 있다.

미디어 쿼리는 다음과 같은 많은 항목을 확인하는데 사용된다

  • 뷰포트의 너비 및 높이
  • 장치(device)의 너비 및 높이
  • 방향(orientation) - 태블릿/휴대폰이 가로모드 혹은 세로모드?
  • 해상도(resolution)
        @media only all and (조건문) {실행문}

        @media not|only mediatype and (mediafeature and|or|not mediafeature) {
            CSS-Code;
        }
  • @media: 미디어쿼리가 시작됨을 표시.
  • not: 전체 미디어 쿼리의 의미를 반전시킴.
  • only: 미디어 쿼리를 지원하지 않는 예전 브라우저에서 지정된 스타일을 적용할 수 없도록 함.
  • and: 미디어 특성을 미디어 유형 또는 다른 미디어 특성과 결합.
  • 미디어 유형(Media Types): all/print/screen/speech
    • all: 기본값. 모든 미디어 유형 장치에 사용
    • print: 인쇄 결과물 및 출력 미리보기 화면에 표시중인 문서
    • screen: 컴퓨터 화면, 태블릿, 스마트폰 등에 사용
    • speech: 페이지를 큰 소리로 '읽는' 스크린리더에 사용
    • 사용하지 않는 미디어 유형: CSS2.1과 Media Queries 3 모듈은 여러가지 추가 유형(tty, tv, projection, handheld, braille, embossed, aural)을 정의했으나 Media Queries 4에서 제거됐으므로 사용해선 안됩니다. aural은 유사한 유형인 speech로 대체됐습니다.
  • 미디어 특성(Media Feature): 사용자 에이전트, 출력 장치, 환경 등의 특징을 나타낸다.
