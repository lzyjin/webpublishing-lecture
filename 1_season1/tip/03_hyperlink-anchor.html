하이퍼링크(Hyperlink)와 앵커(Anchor) 만들기

    <a href = "링크주소 (html파일경로, 이미지 파일경로, #아이디이름"
        target = "탭구분 ( 링크된 파일을 어떻게 띄울지 결정- _blank :새탭으로 보이기, _self : 현재 탭에서보이기 (기본값 ) "
        title = "툴팁(마우스 호버했을 때 보일 텍스트 표시">표시 텍스트/이미지 </a>

앵커만들기 : 한 페이지 안에서 지정된 위치로 점프하여 스크롤하기

    <a href = "#앵커이름"> 표시 텍스트 또는 이미지 <a>
    # : 현재 문서를 의미 

    <태그 id="앵커이름"><태그>

1. a태그의 href속성은 반드시 #로 시작해야한다 #은 현재문서를 의미 즉 #part1은 현재문서(#)에서 part1이라는 id를 찾는다 라는 의미
2. 위로가기 버튼, 상단으로 버튼은 href에 #만 넣으면 된다 클릭하면 문서의 가장 상단으로 이동한다
