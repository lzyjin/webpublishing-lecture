* nmargin, padding 속성의 크기 지정 1. 한 개의 크기 값을 사용 -> 모든쪽(상하좌우)에 동일한 크기 값이 적용 ex) margin : 30px; 2. 두 개의 크기 값을 사용 -> 상하(top bottom), 좌우(left right) ex) margin : 30px 5px; 3. 4개의 크기 값 사용 -> 각각 적용 ( 왼쪽위부터 시계방향으로 ) top, rigt, bottom, left ex) margin : 10px 20px 30px 40px;
