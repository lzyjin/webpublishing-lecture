grid 부모요소 속성

  • column 과 row : column은 세로(열), row는 가로(행)
  • template : 부모요소에 사용.
    gird 레이아웃의 시작은 display: grid; 라는 속성과 값.
    두번째는 grid-template-column으로 세로를 몇개로 나눌지 결정


1) 부모요소에 사용하는 grid 속성 : display: grid; display: inline-grid;

그리드 부모요소를 정의 (해당 선택자에 그리드를 적용한다)

flex의 경우 display:flex;만으로도 자식요소가 가로로 배치된다. 왜냐하면 flex-direction:row를 기본값으로 갖기 때문이다.
하지만 grid는 display:grid만으로는 어떤 변화도 줄 수 없다.

display: inline-grid;는 자식요소의 내용에 맞게 너비가 줄어든다. display:inline-flex;와 동일하다.

2-1) 부모요소에 사용하는 grid 속성 : grid-template-columns: auto | 단위

자식요소 컬럼(열, 세로)을 크기 또는 단위로 정의

auto가 2개이면 자식요소를 컬럼(세로)으로 2분할 한다는 의미. auto가 3개이면 3분할 한다는 의미.
단위 대신에 auto를 사용했기 때문에 알아서 자동으로 1/2 또는 1/3으로 나눈다.
auto 대신 특정 단위를 넣으면 그곳은 고정하고 나머지들은 부모요소의 너비에 꽉 차게 자동으로(auto) 배분한다.

2-2) 부모요소에 사용하는 grid 속성 : grid-template-columns: fr 단위

fr은 fraction, 일부, 파편이라는 뜻으로 flex-grow와 같은 개념

fr의 개수가 auto의 개수처럼 컬럼의 수를 결정하므로, 2개의 컬럼을 만들려면 fr이 2개 있어야 한다.

px % auto fr 단위를 섞어서 사용해도 된다

3-1) 부모요소에 사용하는 grid 속성 : grid-template-rows: auto | 단위

자식요소 row(행, 가로)를 크기 또는 단위로 정의

grid-template-rows의 첫번째 값은 첫번째 행의 높이, 두번째 값은 두번째 행의 높이

행이 2개인데 값을 하나만 적으면 두번째 행의 높이는 auto가 된다.
auto로 적용하지 않으려면 행의 개수만큼 단위를 적어줘야한다.

3-2) 부모요소에 사용하는 grid 속성 : grid: [grid-template-rows] / [grid-template-columns]

그리드 템플릿을 행 / 열 축약형으로 표시

grid를 축약형으로 사용하는것은 권장하지 않는다. 독립적으로 사용하는 것이 훨씬 직관적이고 쉽다.

grid: 100px 200px / auto auto auto;

4) 부모요소에 사용하는 grid 속성 : grid-column-gap, grid-row-gap, grid-gap

자식요소의 가로 또는 세로 사이의 간격을 정의

grid-column-gap: 좌우 여백을 조절, grid-row-gap: 상하 여백을 조절, grid-gap: 좌우여백 상하여백 조절

grid-template-columns: auto auto auto; grid-column-gap: 30px; grid-row-gap: 60px; 또는 grid-gap: 60px 30px;
