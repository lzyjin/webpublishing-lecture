grid 자식요소 속성

1-1) justify-self: start | center | end | stretch

부모요소의 정렬에 관계없이 자식요소를 개별적으로 (컬럼내에서) 수평 정렬

    .parent1 {
      ...
      display: grid;
      grid-template-columns: auto auto;
      justify-items: center;
      align-items: center;
    }
    .parent1 div:nth-child(2) {
      justify-self: end;
    }

1-2) align-self: start | center | end | stretch

부모요소의 정렬에 관계없이 자식요소를 개별적으로 (컬럼내에서) 수직 정렬

    .parent2 div:nth-child(2) {
      align-self: end;
    }

1-3) justify-self, align-self 속성의 기본값은 normal(=stretch)

stretch는 자식요소의 너비값과 높이값이 없는 경우에 default로 좌우 상하로 가득 채운다

2-1) grid-column-start, grid-column-end

그리드 아이템(자식요소)를 배치하기 위해 그리드 선(line)의 '시작위치'와 '끝위치'를 지정한다, '숫자'를 지정하거나 '선이름'을 지정하거나 span 키워드를 사용한다

grid-column-start는 자식요소가 시작하는 열라인, grid-column-end는 자식요소가 끝나는 열라인

    .parent4 div:nth-child(1) {
      background-color: crimson;
      grid-column-start: 1;
      grid-column-end: 3;
    }

2-2) grid-column: 정수 / 정수

grid-column은 grid-column-start와 grid-column-end의 단축 속성

grid-column: [grid-column-start] / [grid-column-end];

가독성을 위해 띄어쓰기 권장

    .parent5 div:nth-child(1) {
      background-color: crimson;
      grid-column: 1 / 3;
    }

2-3) grid-column: 정수 / span 정수

span + 숫자 = 숫자만큼 라인을 확장한다는 개념

명시하지 않으면 span + 1 이 기본값(디폴트로 그리드 한칸을 차지함)

    .parent6 div:nth-child(1) {
      background-color: crimson;
      grid-column: 1 / span 2;
      /* grid-column: 1 / 1 + 2; 와 동일 */
      /* grid-column-start: 1; grid-column-end: 3; 과 동일*/
    }

3-1) grid-row-start, grid-row-end

그리드 아이템(자식요소)를 배치하기 위해 그리드 선(line)의 '시작위치'와 '끝위치'를 지정한다, '숫자'를 지정하거나 '선이름'을 지정하거나 span 키워드를 사용한다

grid-row-start는 자식요소가 시작하는 행라인, grid-row-end는 자식요소가 끝나는 행라인

    .parent8 .item {
      background-color: crimson;
      grid-row-start: 1;
      grid-row-end: 5;
