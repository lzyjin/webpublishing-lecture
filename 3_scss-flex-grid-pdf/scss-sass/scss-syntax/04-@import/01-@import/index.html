외부파일 SCSS로 가져오기 (@import)

@import로 외부 .scss파일을 불러와서 현재 .scss파일에 병합한다.
css 초기화를 위한 reset.scss,
변수를 따로 관리하는 variable.scss,
색상 값을 따로 관리하는 colors.scss,
폰트 관련 코드를 따로 관리하는 fonts.scss
등을 별도 분리된 파일에서 가져올 때 사용한다. 

    문법)
    현재 scss파일안에서
    @import "파일명";
    파일명에는 확장자를 포함시켜도 되고, 확장자를 제외해도 된다.
    
    예시)
    @import "reset.scss";
    @import "reset";
    -------------------------------------------------------
    @import 키워드는 콤마(,)를 이용해서 다중 파일을 한꺼번에 가져올 수 있다.

    예시)
    @import "reset", "variables", "colors", "fonts";
