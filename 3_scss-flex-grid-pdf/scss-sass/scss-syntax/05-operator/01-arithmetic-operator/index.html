scss에서 연산 사용하기 - 산술연산자

scss는 기본적인 연산을 제공한다.
레이아웃 설계 및 HTML 요소의 크기를 일정하게 만들 때 등 연산자를 사용해 다양한 크기 배분을 할 수 있다.

주의

  • 연산에서 +, -, * 는 앞의 숫자에서 뒤의 숫자를 더하거나 빼거나 곱한다. 그리고 변수를 연산 대상으로 사용할 수 있다.
  • css에서 자주 사용되는 calc()는 scss에서 사용할 수 없다.
  • 나누기 연산은 2가지를 주의해야한다
    1. 연산 대상을 괄호로 묶는다
    2. 괄호로 묶지 않으려면 나눠지는 대상이 변수여야한다
    산술연산자

    종류 / 기능
    + : 더하기
    - : 빼기
    * : 곱하기
    / : 나누기
    % : 나머지




    
  
    $fs: 15px;
    $box-margin: 50px;

    .frame {
      width: 50px + 50px;
      height: 100px - 30px;
      font-size: $fs * 2;

      margin: 50px / 2;         // 계산 불가 -> css에서 그대로임
      margin: calc(50px / 2);   // 계산 불가 -> css에서 그대로임
      
      margin: $box-margin / 2;  // 정상 작동 -> css에서 25px
      margin: (50px / 2);       // 정상 작동 -> css에서 25px
    }
