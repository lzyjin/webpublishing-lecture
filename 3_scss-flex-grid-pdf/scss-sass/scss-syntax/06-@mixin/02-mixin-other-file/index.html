scss에서 선언된 @mixin을 @include로 다른 선택자에 사용하기, 독립 파일로 만들어 불러오기

@mixin을 매번 만드는 것보다
예를 들어 mixin.scss 라는 파일을 만들어서 자주 사용하는 그룹을 @mixin으로 여러 개 만들어 놓고,
.scss에서 @import로 불러오면
mixin.scss 파일 안에 미리 세팅된 믹스인들을 활용할 수 있다.

ex) body태그 기본서식은 자주 쓰이니까 별도의 파일로 만들어놓으면 매번 새로 만들지 않아도 되니까 편리하겠죠?
