중첩 - css 속성 단축형으로 사용하기

  • 선택자 내에서 사용한다
  • 선택자 내의 속성에서 공통된 앞글자를 사용해서 단축 속성을 만든다(접두어 형태로 자동 생성)
    scss -----------------------------
    .container {
      font: {
        family: 'Montserrat', sans-serif;
        size: 38px;
        weight: 500;
      }
    }
    는 
    
    css ------------------------------
    .container {
      font-family: 'Montserrat', sans-serif;
      font-size: 38px;
      font-weight: 500;
    }
    가 된다
