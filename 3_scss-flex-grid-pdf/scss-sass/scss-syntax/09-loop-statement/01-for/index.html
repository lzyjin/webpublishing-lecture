반복문 - @for

자바스크립트와 마찬가지로 반복문 for는 변수($변수)를 선언하고 시작(from)해서 종료(through)되기 전까지 반복할 표현식을 계속 반복적으로 출력한다.
변수는 일반적으로 $i 로 선언한다. 

    /* 문법 */
    @for $변수 from 시작 수 through 종료 수 {
      // 반복할 표현식 
    }

    /* 예시 */
    @for $i from 1 through 10 {
      .frame div:nth-child(#{$i}) {
        width: 50px * $i;
      }
    }
