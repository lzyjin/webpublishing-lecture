css 포지셔닝 - 엘리먼트 수평 정렬하기 ( 속성 : float )

float 속성은 요소가 부모요소를 기준으로 왼쪽 또는 오른쪽에 배치할지 지정 float 속성은 인라인, 블록, 인라인블록 모든것에 적용 가능

    float 속성
  • none(default) : 적용하지않음
  • left : 요소가 컨테이너의 왼쪽에 배치
  • right : 요소가 컨테이너의 오른쪽에 배치
  • margin : auto (반드시 block 요소만 가능 )

css 포지셔닝 ( 속성 : clear )

float 속성이 적용되면 다음 요소가 float 속성을 상속받는데 clear 속성이 이것을 해제시킨다

    clear 속성
  • left : 왼쪽으로 배치된 float 속성 상속을 해지
  • right : 오른쪽으로 배치된 float 속성 상속을 해지
  • both : 왼쪽 또는 오른쪽으로 배치된 모든 float 속성 상속을 해지
float : none
float : left
float : right
clear : both

가로배치하는 2가지 방법 - 1. float와 overflow 사용, 2. display:inline-block

1. float와 overflow 사용해서 가로배치하기 단점) 단 1px 도 틀려선 안된다 ㅎㄷㄷ

2. display:inline-block
