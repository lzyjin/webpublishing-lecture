반복문 - @each

자바스크립트와 마찬가지로 반복문 @each는 자바스크립트의 배열(Array) 형태를 사용하는 것이 보통이다. 

    /* 문법 */

    @each $변수 in $변수 {
      // 반복할 표현식
    }
    --------------------------------
    /* 예시 */

    $size: 30px, 60px, 90px, 120px;

    @each $s in $size {
      .logo-#{$s} {
        width: $s;
        height: $s;
      }
    }
