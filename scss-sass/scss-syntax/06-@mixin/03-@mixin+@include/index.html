scss에서 선언된 @mixin과 @include의 혼합형

1. @include로 @mixin을 사용하면서 추가적인 css를 사용할 수 있다.
2. 하나의 @mixin에 여러개의 @mixin을 불러올 수 있다. 

    1.예시 

    @mixin notice-button {
      font-size: 15px;
      width: 120px;
      padding: 7px;
      background-color: #fff;
      cursor: pointer;
    }
    .complete {
      @include notice-button;
      border: 1px solid royalblue;
      color: royalblue;
    }
  
    2.예시

    @mixin loading-msg {
      @include headline;
      @include body-text;
      @include link-text;
    }
