@mixin과 @inclde 사용할 때 컨텐츠를 블록 형태로 전달하는 @content

@mixin 내부에 @content가 있으면 해당 부분의 컨텐츠를(스타일)을 블록 형태로 전달한다.
기존 믹스인 기능에 선택자 또는 속성을 추가 가능 

    /* 문법 */
    @mixin 믹스인이름($...) {

      ..생략..

      @content;
    }
    
    선택자 {
      @include(...) {
        추가할 css 스타일 작성
      }
    }
  
    /* 예시 */
    // scss
    @mixin border-radius($radius) {
      border-radius: $radius;
      @content;
    }
    .box {
      @include border-radius(10px) {
        border-style: dotted;
      }
    }

    // css
    .box {
      border-radius: 10px;
      border-style: dotted;
    }
