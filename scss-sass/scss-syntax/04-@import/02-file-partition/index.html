외부파일 SCSS로 가져와서 파일 분할

프로젝트 규모가 커지면 css 파일의 양이 많아진다.
특히 header와 footer, 그리고 개별 파일들에 css 내용이 많아지기 때문에 파일을 따로 만들어서 관리할 필요가 있다.
(프로젝트 정책에 따라 알맞게 조정할 것.) 

    css폴더
    - style.css
    - header.css
    - footer.css 
    - main.css 
    scss폴더
    - style.scss
    - header.scss
    - footer.scss 
    - main.scss 

    3가지 .scss 파일이 @import된 style.scss파일을 컴파일한 style.css에 모든 css코드가 모이고, html파일에서 style.css를 링크
