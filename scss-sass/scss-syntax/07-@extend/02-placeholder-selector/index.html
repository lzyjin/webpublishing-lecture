@extend로 복사된 선택자를 컴파일된 css코드에서 감추는 플레이스 홀더 선택자 - %

복사 대상 선택자를 표시하지 않는 플레이스홀더(placeholder selector)는 기호 %로 시작한다.

조금 더 깔끔한 css 코드를 작성 할 수 있다. 

    // scss 
    %shape {
      width: 200px;
      height: 100px;
      padding: 10px;
      box-sizing: border-box;
      border: 5px solid #000;
      display: inline-block;
    }
    .frame div:first-child {
      @extend %shape;
      background-color: gold;
    }
    .frame div:last-child {
      @extend %shape;
      background-color: tomato;
    }
  
    // css
    .frame div:first-child, .frame div:last-child {
      width: 200px;
      height: 100px;
      padding: 10px;
      box-sizing: border-box;
      border: 5px solid #000;
      display: inline-block;
    }
    .frame div:first-child {
      background-color: gold;
    }
    .frame div:last-child {
      background-color: tomato;
    }
