변수에 변수 할당하기 (알아만 두기)

변수에 또 다시 변수를 할당할 수 있다. 실용성은 잘 모르겠으니 scss 문법 중 하나라고 알아만 두자. 

    예시)
    
    $crimson: crimson;
    $black: #202020;

    $color-primary:  $crimson;
    $color-secondary: $black;

    .portfolio {
      color: $color-secondary;
      h1 {
        color: $color-primary;
      }
    }
