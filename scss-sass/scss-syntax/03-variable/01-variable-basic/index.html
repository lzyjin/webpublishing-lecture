변수 - 변수(variables) $ 사용하기

변수는 말그대로 변하는 수이다.
반복적으로 사용되는 값을 재활용하기 위해 미리 변수에 저장해 놓고 변수이름을 속성이름 대신 사용한다.
scss에서 변수 키워드 지정은 $기호를 사용한다.
변수를 사용하면 폰트 사이즈, 색상, 크기 등 효율적인 속성 관리가 가능하다. 

    문법)

    $변수명: 값;
    
    선택자 {
      속성: $변수명;
    }

    --------------------
    예시)

    $primary: royalblue;
    $secondary: yellowgreen;
    $btn-padding: 10px 30px;

    .btns {
      text-align: center;
      .btn {
        padding: $btn-padding;
        color: #fff;
        &.primary {
          background-color: $primary;
        }
        &.secondary {
          background-color: $secondary;
        }
