중복 선택자(Duplicate Siblings Selector)

중복 선택자는 태그와 함께 선택자를 중복시키거나, 클래스를 중복시키는 경우를 말한다.
css에서의 p.info {...}, item.focus{...} 와 같은 경우를 scss에서 만드는 방법이다.
이런 경우 중첩(nesting)을 만드는 키워드 &와 함께 사용한다. 

    예시) &.confirm {} = .btn.confirm {}, &#confirm {} = .btn#confirm {}

    .btn {
      padding: 10px;
      text-align: center;
      width: 100%;
      &.confirm {
        background-color: yellowgreen;
      }
      &.warning {
        background-color: crimson;
      }
    }
