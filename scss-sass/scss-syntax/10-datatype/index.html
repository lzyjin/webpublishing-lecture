scss에서 사용하는 데이터 타입

scss에서 사용하는 데이터 타입 = 사용할 수 있는 값
대부분 색상 값, 숫자, 문자를 사용한다. 

    Numbers : 숫자
    Strings : 문자열 
    Colors : 색상 값 
    Booleans : 논리(참과 거짓)
    Nulls : 비어있음
    Lists : 공백 또는 콤마로 구분된 목록 
    내장함수 : ex)색상함수
  
    /* 예시 */
    $this-null: null;

    $this-true: true;
    $this-false: false;

    $this-number: 777;

    $this-string: 'Yejin Zzang';
    $this-string-combine: 'Yejin Zzang' + 'huhuhu';

    $this-color1: crimson;
    $this-color2: #000000;
    $this-color3: rgb(34, 60, 173);
    $this-color4: rbga(34, 60, 173, 0.5);
    $this-color-function: lighten(yellowgreen, 25%);

    .item::before {
      content: $this-string;
      color: $this-color1;
      margin-right: 10px;
    }
    .item::after {
      content: $this-string-combine;
      color: $this-color-function;
      margin-left: 10px;
    }
