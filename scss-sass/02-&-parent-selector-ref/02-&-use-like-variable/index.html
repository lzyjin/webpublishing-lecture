&를 응용해서 변수처럼 사용하기

scss에서는 &(Ampersand)를 응용하면 변수는 아니지만 변수처럼 효율적으로 사용할 수 있다.

폰트사이즈, 색상 등 웹페이지에 자주 사용되는 클래스 네임을 미리 세팅하는 용도로 사용하면 효율적이다 

    예시) & = .font

    .font {
      &-large {
        font-size: 60px;
        color: crimson;
      }
      &-medium {
        font-size: 40px;
        color: yellowgreen;
      }
    }
