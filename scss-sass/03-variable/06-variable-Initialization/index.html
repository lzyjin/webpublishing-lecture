변수의 초기화 - !default (알아만 두기)

!default는 할당되지 않은 변수의 초기값을 설정한다.
쉽게 말해 할당되어있는 변수가 이미 할당되어 있는 변수와 중복된다면 !default를 사용해서 최초에 할당된 변수의 값을 사용한다는 뜻.

보통 부트스트랩, UIKit, Semantic UI 등 CSS Framework를 사용할 경우 해당 프레임워크에서 scss 변수가 세팅되어있다.
작업자가 새로 만든 변수가 CSS Framework의 변수를 덮어쓰기하는 경우를 방지하고자 할 때 사용한다.

주의)) !default와 !important는 다르다. !important는 최우선 순위 속성으로 만드는 기능이고, scss에서도 동일하게 사용한다.
